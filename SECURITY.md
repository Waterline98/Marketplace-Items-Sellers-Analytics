# Безопасность

- **Не коммитьте** файл `.env` и любые файлы с ключами, паролями или токенами. Используйте только `.env.example` как шаблон.
- Секреты задавайте через переменные окружения или секретное хранилище (Airflow Variables, Kubernetes Secrets и т.п.).
- При утечке ключей — немедленно отзовите их и создайте новые.
